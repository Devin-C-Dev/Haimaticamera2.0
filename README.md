# 海马体相机 - Haimatic Camera

> 📸 基于 Web 的拍立得相机应用，支持 AI 图像生成

## ✨ 特性

- 📷 实时相机功能
- 🤖 Coze AI 图像生成
- 🎨 多种滤镜效果
- 📱 响应式设计

## 🚀 部署

### Railway
- 主站：https://haimaticamera.up.railway.app

### Vercel
- 备用：https://haimaticamera-1-0.vercel.app

## 🔧 环境变量

- `COZE_API_TOKEN`: Coze API Token
- `PORT`: 3001

## 📝 技术栈

- Node.js + Express
- HTML5 + CSS3 + JavaScript
- Coze API
